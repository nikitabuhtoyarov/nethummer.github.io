    • Введение

    Развивающийся бизнес требует информатизации. Клиент - сеть станций технического осмотра автомобилей, модель развития - партнерская (сеть ставит на сторонние автосервисы собственное оборудование для прохождения техосмотров, организует юридическую сторону работы).

    Задачи проекта:

    1. Разработать информационную систему управления сетью СТО

    2. Разработать функционал бонусной программы

    3. Интегрировать с SMS-сервисом для рассылки уведомлений клиентам

    4. Разработать шаблоны партнерских инстансов

    Дальнейшие работы:

    - разработка функционала выдачи техосмотров и интеграции с ЕАИСТО

    - разработка реферальой системы привлечения клиентов

    - разработка "под ключ" систем автоматизации для партнеров сервиса и интеграция с онлайн-кассами

    Центральный инстанс

    Из центрального инстанса осуществляется управление бонусной программой и создание партнерских инстансов.

    Администратор видит историю всех заказов у всех партнеров, фильтрует по дате и смотрит общую статистику. Здесь можно увидеть результат работы бонусной программы ТОПСТО.

    Ведется клиентская база. Клиенты получают бонусные баллы, уведомления по СМС о сроках прохождения техосмотра, бонусы за отзывы и поздравление на дни рождения.

    Проведена интеграция с сервисом sms.ru, система шлет по api данные в сервис в ряде случаев.

    Ведется управление партнерами, прочий функционал.

    Система гибкая, построенная по модульно-аспектному принципу. Часть настроек вынесена в отдельный сервисный слой, часть - в виде контроллеров в саму систему. Сервис будет развиваться вместе с франчайзи столько, сколько нужно.



    Аккаунт партнера

    Партнеры имеют свой независимый инстанс, связанный с центральным по api на предмет обмена необходимыми данными. По сути, каждый партнер может развивать свой собственный инстанс отдельно от франчайзи, интегрировать с собственным веб-сайтом (или заказать у нас его разработку), подключить онлайн-кассу (54-ФЗ привет!) и получить в итоге свой кастомный софт для управления бизнесом, связанный с франшизой ТОПСТО.

    Аналогов на рынке нет! Всякая франшиза жестко навязывает собственные корпоративные информационные системы, где франчайзи просто не может ничего настроить под себя.

    Когда потребуется сайт - будет разработан прямо "поверх" системы, бесшовно.

    В результате

    Что получается в результате? Независимые аккаунты партнеров, объединенные в общую сеть, индивидуальная бонусная и реферальная программы, возможность неограниченного развития и персонализации каждого аккаунта в системе и многое другое.

    На базе этого продукта можно разработать аналоги для любой другой сферы с N независимых участников одной сети, будь то торговой или сервисной. По данной технологии можно объединять в сеть филиалы, отделы, предприятия, бизнесы, связывая только нужным функционалом.

    Разработка похожего продукта для любой сферы будет стоить от 200 тыс. рублей, в зависимости от сложности и "навороченности" системы, срок - от 1 месяца.
